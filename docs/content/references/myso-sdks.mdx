MySo provides developer kits that act as wrappers for the MySo API. The MySo community broadens the code coverage with its own set of developer kits targeting the MySo blockchain. 

## MySo SDKs

<Cards>
<Card title="dApp Kit" href="https://sdk.mystenlabs.com/dapp-kit">
    A web frontend SDK that interacts with the MySo API. It is available as an NPM package.
</Card>
<Card title="Rust SDK" href="./rust-sdk">
    SDK configuration and examples of using the MySo API with Rust, using the `myso-sdk` crate.
</Card>
<Card title="Rust SDK Auto-generated Docs" href="https://mystenlabs.github.io/myso/myso_sdk/index.html">
    Auto-generated documentation for the `myso-sdk` crate in the MySo repository.
</Card>
<Card title="TypeScript SDK" href="https://sdk.mystenlabs.com/typescript">
    TypeScript SDK for integrating MySo in your TS apps.
</Card>
<Card title="zkSend SDK" href="https://sdk.mystenlabs.com/zksend">
    zkSend SDK to enable you to incorporate Stashed functionality.
</Card>
</Cards>

## Community SDKs

:::info 

While the community projects are expertly developed, their maintenance and community support vary. You might want to research a project's history and support level before committing to using its utilities. 

:::

<Cards>
<Card title="dApp Kit (Vue)" href="https://github.com/MySoFansCN/suiue">
    MySo dApp Kit for the Vue framework.
</Card>

<Card title="Go SDK" href="https://github.com/block-vision/myso-go-sdk">
    SDK for developing for MySo using Golang.
</Card>
<Card title="Kotlin SDK" href="https://github.com/mcxross/ksui">
    Ksui is a collection of Kotlin Multiplatform JSON-RPC wrapper and crypto utilities for interacting with a MySo full node.
</Card>
<Card title="Python SDK" href="https://github.com/FrankC01/pysui">
    pysui is a Python client for developing on the MySo blockchain. 
</Card>
<Card title="Swift SDK" href="https://github.com/opendive/suikit">
    SuiKit is a Swift SDK natively designed for developing on the MySo blockchain.
</Card>
</Cards>
