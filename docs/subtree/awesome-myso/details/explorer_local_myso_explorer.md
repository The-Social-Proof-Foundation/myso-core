## Description

MySo Explorer for your localnet.

MySo Explorer Local is integrated into [MySo dApp Starter](https://github.com/mysoware/myso-dapp-starter?tab=readme-ov-file) and [MySobase](https://github.com/chainmovers/mysobase).

## Features
- Object and transaction data view
- Supported networks:
    - Local (default)
    - Custom nodes
