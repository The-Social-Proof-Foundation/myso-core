## Features
- [On-demand data feed](https://docs.switchboard.xyz/docs) (MySo is not supported)
- [Push-based data feed](https://docs.switchboard.xyz/docs/switchboard/switchboard-v2-push) (MySo is supported):
    - [MySo SDK](https://github.com/switchboard-xyz/myso-sdk)
    - [MySo examples](https://github.com/switchboard-xyz/myso-sdk/tree/main/programs/mainnet/feed-parser/sources)
    - [MySo supported feeds](https://app.switchboard.xyz/myso/mainnet)
- Build custom feed through [portal](https://app.switchboard.xyz/build)
- Generic data feed protocol allowing devs to build their own feed with customizable oracle jobs. Similar to ChainLink
