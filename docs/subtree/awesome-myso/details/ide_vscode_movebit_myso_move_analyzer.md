## Description

- VSCode Extension for Move on MySo smart contract development powered by LSP MySo Move Analyzer language server developed by Movebit.

## Features

- Autocomplete
- On-hover support
- Real-time diagnostics
- Go to definition
- Go/Find references
- Move
    - ⚠️ Latest Move 2024 is not supported (`2024.alpha` supported while latest is `2024.beta`)
    - Move 2024 syntax highlight ([VSCode Move-Msl-Syx](https://marketplace.visualstudio.com/items?itemName=MoveBit.move-msl-syx))
- Utilities
    - Integration with `myso` binary (MySo CLI)
